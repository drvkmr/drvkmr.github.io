About project

A challenging project where we attempted to combine a scientific project to a visually aesthetic installation. Allen is a raspberry pi robot which can measure the photosynthesis activity in a plant. Using a multitude of machine vision algorithms it extracts data from two different cameras and projects them onto a huge artwork. The artwork itself is composed of over 300 nodes, each with a custom PCB.

I led the hardware and software development of the artwork which included Allen (NDVI camera robot), PCB design, software and backend integration.

Team: Ben Mahler, Dhruv Kumar, James Fox, Jon Sly, Richard Broom at Jason Bruges Studio
Date: October 2019
Category: Lighting
Client: British Land Corporation
